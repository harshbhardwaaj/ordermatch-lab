# OrderMatch Lab

AI workbench for extracting purchase orders, matching messy line items to SKUs, and evaluating order automation quality.

The project is aimed at the real bottleneck in B2B order automation: turning messy customer orders, RFQs, emails, PDFs, and product catalogs into ERP-ready structured data with measurable confidence.

## Initial Focus

- Extract structured order data from semi-structured purchase orders and RFQs.
- Match noisy product descriptions to internal SKUs.
- Route uncertain matches to human review.
- Measure extraction and matching quality with repeatable evals.
